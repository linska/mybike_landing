# MyBike Landing Page

A responsive landing page for a modern bike brand, built as a layout practice project.

## Demo

[Live Demo](https://linska.github.io/mybike_landing/)

## Technologies

- HTML5
- SCSS / Sass
- JavaScript
- Parcel
- Mate Academy scripts

## Features

- Responsive layout for mobile, tablet, and desktop
- Mobile menu with open/close behavior
- Menu closes on navigation link click and Escape key
- Header style changes on scroll
- Product comparison section
- Details section with image gallery
- Contact form with basic HTML validation
- Smooth UI interactions and hover effects
- BEM-style class naming

## Getting Started

### Clone the repository

```bash
git clone https://github.com/linska/mybike_landing.git
cd mybike_landing
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm start
```

### Build for production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```


## Features
* **Responsive Design** — optimized for mobile, tablet, and desktop devices
* **Smooth Navigation** — anchor-based navigation with a CSS-powered mobile menu
* **Product Showcase** — featured products section with interactive hover effects
* **Category Gallery** — visually engaging category grid with smooth scale transitions
* **About Section** — immersive brand story with a full-width background image
* **Contact Form** — user-friendly form with validation and automatic reset after submission
* **Modern Typography** — clean and elegant design powered by the Manrope font family
